About This Page

This page shows how contact information can be organized using semantic HTML. Instead of using layout tricks, the information is placed inside a table because it represents structured data. Each row shows a different way to contact me.

If this website were expanded in the future, this page would include a contact form so visitors could send messages directly. For this assignment, we are focusing on structure and meaning rather than building a working form.
